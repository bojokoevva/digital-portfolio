AI Image Recognition – Teachable Machine Demo

A browser-based AI demo that uses Teachable Machine and TensorFlow.js to recognize everyday objects (phone, headphones, person) from the webcam and display class probabilities in real time.

Computer vision Teachable Machine TensorFlow.js JavaScript
ISM 3232 – App Development
University of South Florida · Front-end AI demo

Context & goal

This project started as an experiment in making machine learning feel tangible and personal. Instead of working with abstract datasets, I trained a Teachable Machine model on objects from my own environment: my iPhone, JBL headphones, and myself on camera.

The goal was to build a small but complete end-to-end AI image recognition demo: train a custom classifier, connect it to a web interface with TensorFlow.js, and visualize predictions in a way that is clear for non-technical users.

How it works

The system uses a Teachable Machine image model exported to TensorFlow.js and loaded directly in the browser. The user enables the webcam, shows an object (phone, person, headphones), and the app continuously classifies frames and updates the UI.

  • Collected and labeled images for three classes: iPhone, Ayanka (me), JBL headphones.
  • Trained a Teachable Machine image classifier and exported the model files for web deployment.
  • Wrote vanilla JavaScript to load the model, attach it to the webcam stream, and call predict() on each frame.
  • Displayed the top prediction and confidence scores so users can see how sure the model is and when it gets confused.

UI & interaction

I kept the demo simple so that the focus stays on the AI behavior rather than on heavy UI. The app has a clear “start” interaction and a live prediction area that updates as you move objects in front of the camera.

  • Clean layout with a live video feed, prediction label, and a list of class probabilities.
  • Labels match the real-world objects on my desk, which makes the demo easy to understand for students or non-technical users.
  • Code is small and readable, making this project a good teaching example for how to integrate pre-trained Teachable Machine models.

What I learned

This project helped me connect machine learning concepts with front-end development. I practiced thinking not just about model accuracy, but also about how users understand AI predictions and what makes a demo feel trustworthy and transparent.

  • Learned how to go from a no-code ML tool (Teachable Machine) to a production-like browser integration with JavaScript.
  • Understood the limitations of small custom models, especially how lighting, background, and camera angle affect predictions.
  • Confirmed that I enjoy building interactive AI demos that can be used in teaching, workshops, or future research projects.
