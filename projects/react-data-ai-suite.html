Suite overview

Instead of treating each assignment as an isolated exercise, I approached these projects as a connected suite of front-end applications. Together, they show how I move from UI design and components to charts and, finally, to integrating a machine learning model in the browser.

The three projects include a Tour Explorer for filtering multi-day trips, a Funding Tracker for visualizing startup investments over time and by industry, and an AI Image Recognition Demo powered by Teachable Machine.

Tour Explorer – React travel gallery

Tour Explorer is a React app that presents curated multi-day tours as cards. Users can browse destinations, filter trips, expand descriptions, and mark tours as “not interested,” which removes them from the list.

  • Built reusable card components with props for title, image, price, summary, and actions.
  • Implemented stateful logic for filtering destinations and tracking which tours the user hides.
  • Focused on a clean, friendly layout with strong typography and clear call-to-action buttons.

Funding Tracker – data visualization dashboard

Funding Tracker is a front-end dashboard that visualizes startup funding over time and across industries. It is designed to look like a lightweight analytics tool that an investor or analyst could use to understand trends at a glance.

  • Built bar and line charts to compare total funding by year and by industry segment.
  • Practiced structuring data for charts and keeping the configuration readable and easy to extend.
  • Used soft colors and spacious layout to mimic modern analytics dashboards.

AI Image Recognition Demo – Teachable Machine

The third piece in this suite is an AI image recognition demo built with Teachable Machine and TensorFlow.js. I trained a custom model on objects from my own environment (iPhone, JBL headphones, and myself) and connected it to a simple web interface.

  • Collected and labeled training images for three real-world classes.
  • Exported the Teachable Machine model and loaded it in the browser with JavaScript and the webcam API.
  • Displayed live predictions and confidence scores to make the model’s behavior understandable to non-technical users.
